
# Prompt: Environment Configuration

**Goal**: Define environment variables and config files.

**Instructions**
- Node `.env`: `API_BASE_URL`, `GA_ID`, `PORT`.
- Spring `application.yml`: `data.path` for JSON directory, JWT secret, server port.
- Create `.env.example` and sample `application.yml`.

**Acceptance Criteria**
- Local dev works after copying examples; no secrets in repo.
