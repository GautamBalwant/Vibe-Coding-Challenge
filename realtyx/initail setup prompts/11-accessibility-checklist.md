
# Prompt: Accessibility (AA) Pass

**Goal**: Ensure WCAG 2.1 AA for key pages.

**Instructions**
- Use semantic HTML in EJS; label all inputs.
- Provide keyboard navigation, focus states, skip-to-content.
- Alt text for images; ARIA for complex components.
- Run axe checks and correct violations.

**Acceptance Criteria**
- No critical axe violations; manual keyboard testing passes.
